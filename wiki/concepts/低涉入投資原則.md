## 一句話定義

投資應是「**低時間佔比、以原則驅動的副業能力**」，而非情緒化、頻繁決策的主業——否則財務與人生穩定性將一起被市場波動綁架（[[raw/3-4 聰明投資絕不燒腦.md]]）。

## 身份選擇：逐浪手 vs. 資本配置者

| | 逐浪手（短線 / 當沖 / 技術派） | 資本配置者（低涉入 / 原則派）|
|-|----------------------------|-----------------------------|
| 決策依據 | 市場情緒、K 線、消息面 | 企業基本面、現金流、護城河 |
| 頻率 | 日內 / 週內，心智全占 | 定期檢視，非交易日不看盤 |
| 收益來源 | 流動性噪音、短期波動差價 | 企業長期價值成長 × 時間複利 |
| 黑天鵝暴露 | 極高（槓桿 + 集中 + 高頻） | 極低（分散 + 長持 + 低槓桿）|
| 代表結局 | 牛市造神、熊市清算 | 「傻瓜富翁」（[[wiki/entities/波克夏海瑟威.md]]）|

**立場**：我們不是逐浪手，我們是以現金流與價值為準繩的資本配置者（[[raw/3-4 聰明投資絕不燒腦.md]]）。

## 把投資當主業的結構性謬誤

- **機制失效**：高頻進出放大交易成本（手續費 + 稅 + 滑價）；盈虧受制於流動性風向，而非企業內在價值——收益來源從資本報酬退化為情緒勞動（[[raw/3-4 聰明投資絕不燒腦.md]]；呼應 [[wiki/concepts/時點預測謬誤.md]]）。
- **情緒迴圈**：賺 → 亢奮 → 加碼；虧 → 懊悔 → 復仇式加碼——風控在情緒循環中徹底失效。
- **行為外溢**：準女婿案例——牛市不報喜，熊市資金回撤後情緒外溢至伴侶關係；高壓投機不只傷財，也傷關係（[[raw/3-4 聰明投資絕不燒腦.md]]）。
- **「行業內專長 ≠ 投資勝率」**：案例——證券公司總經理沉迷技術程式，實測能抓短暫波動，但金融海嘯時全線腰斬；業界資歷不等於複利能力（[[raw/3-4 聰明投資絕不燒腦.md]]）。

## 低涉入的操作節律

- **非交易日不看走勢圖與財經節目**：每週一個固定時段檢視投資筆記與持倉，其他時間心智歸還給主業與生活。
- **停止接觸以 K 線「預言」為主的內容**：改看年報、產業數據、企業訪談——資訊品質決定決策品質。
- **低頻再平衡**：每季一次資產配置檢視（持有比例 / 再投入比率 / 是否觸發預設規則），不做行情判斷。
- **書面化原則**：事先寫下「買入 / 持有 / 退出」的觸發條件，執行時不需要重新思考——用規則取代臨場情緒。

## 可複製優勢的來源

真正可持續的投資優勢來自：
1. **基本面理解**：找 5 檔在自己理解圈內的公司，讀近 3 年年報，記錄護城河 / 現金流 / 估值區間。
2. **時間**：讓正確的標的替自己工作，不要搶走標的的「工作機會」（頻繁進出 = 炒掉自己的最佳員工）。
3. **不燒腦的指數化**：對大多數人，[[wiki/entities/0050.md]] 等市值型 ETF 的定期定額 = 用最低時間成本搭上整體市場的複利（[[wiki/concepts/先存後花.md]]）。

## 百億門檻論

> 「若擁有超級安全邊際（百億家底）才能承擔長期投機錯誤；一般人應以風險承擔能力匹配策略複雜度。」（[[raw/3-4 聰明投資絕不燒腦.md]]）

普通人沒有試錯的資本緩衝——一次系統性風險就可能清零，必須走低涉入 + 分散化的路。
