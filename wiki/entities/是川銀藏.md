## 一句話定義

日本傳奇投資人（1897–1992），一生三度白手起家、三度破產，最終 95 歲在破產中逝世——**擁有卓越市場判斷力卻無法知行合一的悲劇典範**。

## 關鍵事實

### 人生軌跡（時序）

| 年份 | 事件 |
|------|------|
| 1897 | 出生於明治維新後沒落的武士家庭 |
| 1911 | 14 歲離家至貿易行工作 |
| 一戰期間 | 20 歲在中國大連靠軍需生意賺取巨富，隨戰亂歸零（**第一次破產**） |
| 1923 | 關東大地震：囤積鐵皮與釘子大賺，同時捐出半數物資行善 |
| 1927 | 工廠在金融恐慌中倒閉（**第二次破產**） |
| 1927–1931 | 躲進圖書館苦讀三年經濟學與統計學，研究景氣循環 |
| 1931 | 以 70 日圓重返市場，年底靠槓桿操作變為 7,000 日圓（暴增百倍） |
| 1938 | 赴朝鮮開辦礦山與製鐵廠，事業版圖擴至萬人規模 |
| 1945 | 日本戰敗，韓國資產全被沒收，再度歸零（**第三次破產**） |
| 1965 | 投資大阪市郊荒地，獲利 3 億日圓 |
| 1966 | 押注水泥股，一年內 3 億→30 億日圓 |
| 1980 | 83 歲押注住友金屬礦業 5,000 萬股（200 日圓→1,200 日圓），繳納近 29 億日圓所得稅，成為**全日本首富**，被譽為「股神」 |
| 1980 年代 | 為救罹癌長子，情感性押注生技股（新藥開發失敗），慘賠 100 億日圓；違背自訓借錢加碼攤平 |
| 1990 | 日經指數腰斬，無法補繳保證金被斷頭，陷入惡性循環 |
| 1992 | 95 歲在破產中逝世，子孫拋棄繼承（繼承來的只有債務） |

### 他自己的投資法則（卻未能執行）

- 「買進要迅速，賣出要冷靜」
- 「七成獲利即賣」（1980 年住友案有做到，但之後打破）
- 「不碰融資（槓桿）」（生技股崩跌後違背此訓）
- 市場永遠有風險

### 致命決策的結構

1. **情感凌駕理性**：為長子的癌症押注生技股，動機是親情而非基本面
2. **違背自訂規則**：股價下跌後借錢加碼，直接違反「不碰融資」的自我法則
3. **時機最壞**：1990 年日經指數大崩跌（12,000 點以上跌至 2,000 多點）遇上強迫斷頭

> 這不是「他不懂投資」，而是「他知道卻做不到」——**知行不一**才是真正的殺手。[[raw/4-1 投資大咖下場怎麼這麼難看？.md]]
