## 一句話定義

德裔匈牙利投資人（1906–1999），兩度破產後悟道轉型，以 93 歲高齡留下約 4,000 萬歐元資產辭世——**少數真正從投機走向智慧、並以富裕晚年收場的傳奇人物**。

## 關鍵事實

### 人生軌跡（時序）

| 年份 | 事件 |
|------|------|
| 1906 | 出生於布達佩斯富裕家庭 |
| 1926 | 20 歲被送往巴黎從事證券業 |
| 1929 | 提前看空美股大賺，成為百萬富翁；美國延期德國戰爭賠款消息出現，空單爆倉（**第一次破產**） |
| 1932 | 押注「瑞典火柴大王」公司債，違約導致（**第二次破產**） |
| 1940 | 德軍入侵法國，逃往紐約，躲過納粹迫害 |
| 美國期間 | 大量低價收購沙皇時期俄國國債（市場視為廢紙） |
| 戰後 | 沙皇國債因國際協商獲償還，「廢紙」價值連城；同期看準德國戰後重建趨勢，投資德國股市 |
| 1950 年代起 | 人過半百後，投資視角從短期投機轉向宏觀景氣循環長線押注 |
| 晚年 | 成為德國電視受歡迎的股市評論家與哲學家 |
| 1999 | 93 歲辭世，留下約 **4,000 萬歐元**資產（2026-04-29，[[raw/4-1 投資大咖下場怎麼這麼難看？.md]]） |

### 核心投資哲學

**「主人與狗」理論**：
- **主人** = 經濟基本面（穩步前行）
- **狗** = 股價（圍繞主人來回奔跑，路程遠多於主人）
- 問題：「你要當人還是當狗？」——試圖捕捉每次波動的人就是那隻疲於奔命的狗；聰明的投資者只需跟隨經濟長期趨勢。

[[raw/4-1 投資大咖下場怎麼這麼難看？.md]]

**「安眠藥投資法」**：
- 買入優質資產後長期持有，「不是長時間的死守，是長時間活著」
- 強調風險管理：在不可預測的市場中**先求生存**
- 講者認為若科斯托蘭尼活在現代，他很可能推薦 ETF

**成功三要素**：耐心、資本、正確的心理準備

**「投機者」的本義**（德文語境）：
- 具備專業判斷、洞悉時勢、承擔相應風險的人
- ≠ 盲目衝動的賭徒
- 他自稱「投機者」，但成熟期理念與巴菲特價值投資並無二致
